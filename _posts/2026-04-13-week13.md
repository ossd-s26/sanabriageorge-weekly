## Open Source But Not Just Code
This week we learned about how open source does not have to be confined to only code. The concept of open source can be applied to any other shared information. Specifically for this week, I chose to do research on Project Gutenberg. Project Gutenberg is an open access digital library that was created in 1971 and still is ongoing. This library provides free public domain books to anyone which removes the financial and accessibility barriers. Currently there are access to over 70,000 books. The way to contribute to this open source project is to sign up as a volunteer in order to help proofread and format books into file types. Unlike Open Library which is similar, Project Gutenberg ensures that you own the book rather than borrowing due to the fact that it stores public domain books. Overall I find it fascinating how there are so many open source projects that exist today that is not just code.
